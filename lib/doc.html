First Question : Explain two reasons why Flutter is considered a good choice for cross-platform development.

Two reasons why Flutter is considered a good choice for multi-platform development :

  • 1. Single Codebase for Multiple Platforms Flutter allows developers to write one codebase that can run on Android, iOS, web, desktop, and embedded devices. This reduces development time, cost, and maintenance effort because developers do not need to write separate apps for each platform.

  • 2. Fast Development with Hot Reload Flutter provides a feature called Hot Reload, which allows developers to instantly see changes in the app without restarting it. This speeds up development, testing, and debugging, making the development process more efficient.

Second Question :

What programming language does Flutter use? Flutter uses the Dart programming language. Why Dart was chosen for Flutter :

  • 1. High Performance Dart can compile directly to native machine code, which helps Flutter apps run smoothly with high performance.

  • 2. Fast Development Features Dart supports features like Hot Reload, making it easier for developers to quickly test and update their code.

  • 3. Easy to Learn Dart has a simple syntax similar to languages like Java and JavaScript, making it relatively easy for developers to learn.

Fifth Question :

What is the difference between List and Set in Dart? :

Answer :

A List is an ordered collection of elements. You can access items using their index.

              void main() {
                List numbers = [10, 20, 30, 20];
                print(numbers);
                print(numbers[1]);
              }

              // Output: 
              // [10, 20, 30, 20]
              // 20

A Set is a collection of unique elements. It automatically removes duplicate values.

              void main() {
                Set numbers = {10, 20, 30, 20};
                print(numbers);
              }

              // Output : 
              // {10, 20, 30}
